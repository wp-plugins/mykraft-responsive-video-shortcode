==Description==

Mykraft Responsive Video Shortcode Plugin is a simple way to add responsive video to your WordPress pages using editor button that inserts responsive video short-code.

 
==Features==

1. TinyMCE editor button shortcode
2. Responsive video container

 
==Installation==

First download the ZIP file
Log in to your website administrator panel
Go to Plugins page, and add new plugin
Upload plugin
Click Install Now button
Then click Activate Plugin button

 
==Frequently Asked Questions==

= Is there any theme restriction for this plugin? =

No, you can use this plugin with any WordPress theme.
